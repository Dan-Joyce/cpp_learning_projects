# Jour 6 – Étude statistique et corrélation animale (C++)

Ce projet illustre une application concrète de la **programmation C++** dans le domaine de la **Data Science**.  
L’objectif est de **générer un ensemble de données animales simulées** (2000 enregistrements) et d’en **analyser les relations statistiques** : moyenne, variance, écart-type et corrélation.

---

## Objectifs pédagogiques

- Manipuler des **structures de données** (`struct`, `vector`, etc.).
- Automatiser la **génération de jeux de données réalistes**.
- Calculer les **mesures statistiques descriptives** :
  - Moyenne
  - Variance
  - Écart-type
- Étudier la **corrélation** entre variables (par ex. poids ↔ espérance de vie).
- Organiser le code en **modules** : génération, analyse, export.
