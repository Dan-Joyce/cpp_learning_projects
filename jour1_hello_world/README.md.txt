#  Jour 1 – Hello World

Ce projet est mon premier programme C++  
Il affiche simplement un message dans la console.

##  Objectif
- Découvrir la structure de base d’un programme C++
- Apprendre à compiler et exécuter avec Code::Blocks
- Tester l’affichage avec `cout`

##  Code source
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Bienvenue dans ton premier programme C++ " << endl;
    cout << "Hello World !" << endl;
    return 0;
}
